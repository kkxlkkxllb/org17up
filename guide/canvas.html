# 画布大小

默认尺寸为 900x506 px，遵循 16:9 比例，后期将支持定义扩大画布

# 画布背景

默认为白色背景，支持背景色翻转为黑板颜色，该设置影响当前所有页面的背景色，请留意

如翻转背景色，需要相应的去调节下笔的颜色，来配合黑板的颜色

# 课件导入

支持导入PDF课件为画布背景，此时画布原背景将被覆盖

支持单个PDF文件最多100页的导入，导入后画布高度将自适应课件高度，超出录制范围部分可以滚动

同时支持导入单张图片作为背景

WARNING

PDF文件导入过程在客户端完成，文件页数越多，导入耗时越长，请耐心等待

# 笔迹灵敏度

笔迹灵敏度可调节，1 - 8，数值越小越灵敏，适用于不同书写场景

如手写速度较快场景，则可将灵敏度调较低值，使得笔迹细节不丢失

默认值为4，正常情况下可满足一般手写输入速度的需求

# 导出到本地

如果录制完以后，不想立即发布微课，可以选择导出本次录制数据到本地暂存

之后可在“我的微课”页面中，通过“上传本地微课”选择再次预览及发布
