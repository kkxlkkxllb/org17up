# 介绍

手绘微课现在已经升级到Pro，功能上聚焦于“高质量手写笔迹过程与语音的同步录制” + “稳定高清省流量跨平台的播放能力”

TIP

支持离线断网状态下继续录制，免受网络不稳定影响

# 语音录制
